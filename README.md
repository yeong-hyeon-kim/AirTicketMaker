# 📕 Air Ticket Maker

> 나만의 항공권을 만들어 봅니다.

## 🏷️ 기능(Function)

- [나만의 항공권 만들기](#기능)

## 💡 사용 예제(Usage Example)

- 항공사(AirLine)
- 좌석 등급(Seat Class)
- 라운지 등급(Lounge Class)
- 편 명(Flight)
- 이름(Name)
- 출발지(From)
- 도착지(To)
- 탑승 시간(Boarding Time)
- 도착 시간(Dept Time)
- 탑승 일자(Date)
- 좌석(Seat)
- 게이트(Gate)

항목을 수정하여 나만의 항공권을 만듭니다.
